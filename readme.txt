Show a carousel that displays your latest posts.

== Description ==

Various settings for number of columns and posts to display.

== Installation ==

1. Upload purely-post-carousel.zip via the plugin upload menu, or use the built-in admin panel to locate "Purely Post Carousel"
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widget areas where to show the widget, and input settings.
